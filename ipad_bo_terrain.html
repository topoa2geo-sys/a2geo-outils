📁 Dossier affaire

Format : AAAA_NNN  (ex : 2026_099)

🏕️ Documents à préparer pour le terrain

📶 Avec connexion — Ouvrir dans Adobe Reader

Les fichiers obligatoires sont pré-sélectionnés. Cochez les pouvoirs DMPC si nécessaire, puis appuyez sur le bouton.

⚠️ Avant de télécharger
Le dossier doit être créé manuellement au préalable dans Fichiers › Sur mon iPad › A2GEO.
✈️ Sans connexion — Télécharger le ZIP avant de partir
Télécharge tous les fichiers de 2-prepa_BO en un ZIP.
Sur iPad :
  1. Créer d'abord le dossier dans Fichiers › Sur mon iPad
  2. Appuyer sur 📦 Télécharger tout en ZIP
  3. La feuille de partage iOS s'ouvre → Enregistrer dans Fichiers
  4. Naviguer vers le dossier et confirmer
  5. Appuyer sur le ZIP pour le dézipper sur place

📤 Retour terrain → Drive

Sélectionnez les fichiers signés sur l'iPad. L'outil les renomme (_sign / _sign_a-verifier) et les uploade dans › 3-SCANS_terrain sur Drive.
