✈️ Sans connexion — Télécharger le ZIP avant de partir
Télécharge tous les fichiers de 2-prepa_BO en un ZIP.
Sur iPad, après le téléchargement :
  1. Ouvrir Fichiers › Onglet Sur mon iPad
  2. Créer le dossier (maintenir appui › Nouveau dossier)
  3. Déplacer le ZIP terrain_.zip dans ce dossier
  4. Appuyer sur le ZIP pour le dézipper
